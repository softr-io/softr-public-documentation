# Linear integration

Connect Linear with your Softr applications to turn app activity into tracked work. Build workflows that create Linear issues automatically — from form submissions, new sign-ups, or record changes — so every request lands in the team's queue without anyone copying it over by hand.

## Overview

The Softr Linear integration links your no-code apps directly to your Linear workspace. Whenever something happens in Softr — a customer reports a bug, a record gets flagged, a new request comes in — you can automatically create a Linear issue in the right team, with the title, description, assignee, priority, status, labels, project, and due date filled in from your workflow.

Whether you're building a client portal, an internal operations tool, or a support dashboard, the Linear integration closes the gap between what happens in your Softr app and the work your team plans and ships in Linear.

## Available Actions

### Create issue

Create a new issue in a Linear team with a title, description, assignee, priority, status, labels, project, and due date — all set dynamically from your Softr workflow. Teams, assignees, statuses, labels, and projects are chosen from live dropdowns pulled straight from your connected Linear workspace.

## Key Benefits

- **No-code simplicity:** Configure every Linear action visually in Softr — no code required beyond the one-time API key setup.
- **Instant issue creation:** Turn any Softr form submission or record event into a Linear issue the moment it happens.
- **Always-current options:** Teams, assignees, statuses, labels, and projects load directly from your Linear workspace, so you never hardcode IDs.
- **Right work, right team:** Route each issue to the correct team, project, and assignee automatically, based on your Softr data.
- **Ops-ready portals:** Build intake forms and internal tools in Softr that feed work straight into Linear, where your product team already plans its cycles.

## How to Connect Softr with Linear

1. Open your Softr workspace and go to **Workflows**.
2. Create a new workflow or open an existing one, then add a Linear action.
3. In the **Account** field, click **Add another account** and paste your Linear **personal API key**.
4. To create an API key, open Linear and go to **Settings → Security & access → Personal API keys**, then click **New API key**.
5. Click **Save** — Softr verifies your key and connects the account.
6. Pick the **team** for the issue, map the remaining fields from your Softr forms or previous workflow steps, then activate your workflow.
