<Note>Scatter chart block is available for users on the [Professional plan and above](https://www.softr.io/pricing).</Note>
<Frame caption="Scatter Chart">![Scatter Chart](/building-blocks/images/type-scatter-chart/image1.png)</Frame>

The **Scatter** type is similar to the [Line Chart](/building-blocks/vikC2AWEpQGkZd4jGyoVxo/type-line-chart/sr4V2LjKyhGdzbmAmEcwwy) with the only difference that there are no connecting lines between the nodes.
