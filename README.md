# Softr Public Documentation

## Production

Live docs: https://docs.softr.io/

Deployment is automatic — merging a PR to `main` triggers a deploy.

## Development

This project uses [Mintlify](https://www.mintlify.com/docs/quickstart) for documentation.

### Running locally

Install the CLI globally:

```bash
npm i -g mint
```

Then start the dev server from the repo root:

```bash
mint dev
```
