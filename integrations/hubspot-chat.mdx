## Creating a Chatflow

To add a chat widget to your Softr app, you first need to create a chatflow in your HubSpot account to be able to receive messages. To do that, go to **Conversations => Chatflows => Create chatflow.**
<Frame caption="Creating a Chatflow">![Creating a Chatflow](/integrations/images/hubspot-chat/image1.png)</Frame>

In the next step, choose **Website**.
<Frame caption="Choosing the “Website” option">![Choosing the “Website” option](/integrations/images/hubspot-chat/image2.png)</Frame>

Then select **Live chat**.
<Frame caption="Selecting the “Live chat”option">![Selecting the “Live chat”option](/integrations/images/hubspot-chat/image3.png)</Frame>

Next, select the **Inbox** and a **Language** for the chatflow, and hit **Create**. You can find more info on configuring these and other parameters of the chatflow [here](https://knowledge.hubspot.com/chatflows/create-a-live-chat).
<Frame caption="Selecting Inbox and Language">![Selecting Inbox and Language](/integrations/images/hubspot-chat/image4.png)</Frame>

Lastly, you enter the edit page where you can [customize](https://knowledge.hubspot.com/chatflows/create-a-live-chat) your chatflow, and, as soon as you are done, enable it in the upper right corner of the page.
<Frame caption="Enabling chatflow">![Enabling chatflow](/integrations/images/hubspot-chat/image5.png)</Frame>

So, your chatflow is ready, and you can go ahead and add the tracking code in your Softr app to start displaying the chat widget to your visitors.

## Getting the Tracking Code

Go to **Settings** in your HubSpot account.
<Frame caption="Accessing settings">![Accessing settings](/integrations/images/hubspot-chat/image6.png)</Frame>

Scroll down to **Tracking Code** and copy the **Embed code**.
<Tip>The location of the tracking code can be different depending on your HubSpot subscription plan. Check out this [article](https://knowledge.hubspot.com/reports/install-the-hubspot-tracking-code) for more details.</Tip>
<Frame caption="Getting the tracking code">![Getting the tracking code](/integrations/images/hubspot-chat/image7.png)</Frame>

## Adding the tracking code in Softr studio

In Softr Studio, go to **Settings => Integrations => HubSpot Chat**, and enter the **Tracking Code**.
<Frame caption="Adding the tracking code in Softr">![Adding the tracking code in Softr](/integrations/images/hubspot-chat/image8.png)</Frame>

Save the **Settings**, republish your app, and you're done. Now, the chat widget should start appearing on your site.
<Frame caption="HubSpot chat widget added">![HubSpot chat widget added](/integrations/images/hubspot-chat/image9.png)</Frame>
